# TodoList made with Reactjs (Microverse project)

This is an educational project for making a todo list web app using Reactjs.

## Built With

- Reactjs

## Demo

A live demo of this app can be seen [here](https://ahangarha.github.io/MV-React-ToDo/).

![Screenshot](./screenshot.png)

## Usage

To build the project run:

```
$ npm install
$ npm run build
```

## Development

In development environment use the following command:

```
$ npm start
```
